# Space Math

Математичка игра у свемирској теми за децу. Вежбај сабирање и одузимање сам, или се такмичи са другом преко кућне мреже.

---

## Шта ти треба

- Компјутер (Mac, Windows или Linux)
- Интернет конекција (потребна само једном, за инсталацију)

---

## Како да инсталираш

### Корак 1: Инсталирај Node.js

Node.js је бесплатан програм који покреће игру. Треба да га инсталираш само једном.

1. Иди на **https://nodejs.org**
2. Кликни на велико зелено дугме на коме пише **"LTS"** (препоручена верзија)
3. Отвори преузету датотеку и прати инсталер — само кликћи **Next** / **Continue** док се не заврши
4. Рестартуј компјутер после инсталације

### Корак 2: Преузми игру

1. Иди на ову страницу: **https://github.com/njelic04/space-math**
2. Кликни на зелено дугме **"Code"**
3. Кликни на **"Download ZIP"**
4. Нађи преузету ZIP датотеку (обично је у фолдеру **Downloads**) и распакуј је
   - **Mac**: Двокликни на ZIP датотеку
   - **Windows**: Десни клик на ZIP датотеку и изабери **"Extract All"**

### Корак 3: Инсталирај додатке за игру

Ово треба да урадиш само једном после преузимања.

1. Отвори терминал:
   - **Mac**: Отвори апликацију **Terminal** (претражи је у Spotlight са Cmd + Space)
   - **Windows**: Отвори апликацију **Command Prompt** (претражи је у Start менију)
2. Укуцај следеће и притисни Enter (најлакше је да копираш и налепиш):
   ```
   cd Downloads/space-math-main
   ```
   > Ако си преместио фолдер негде другде, замени путању горе са оном где си га ставио.
3. Укуцај ово и притисни Enter:
   ```
   npm install
   ```
   Сачекај неколико секунди док се заврши.

## Како се игра

### Покрени игру

1. Отвори терминал (исто као у Кораку 3 горе)
2. Иди у фолдер игре:
   ```
   cd Downloads/space-math-main
   ```
3. Покрени игру:
   ```
   npm start
   ```
4. Требало би да видиш поруку попут:
   ```
   Space Math Server Running!
   Local:   http://localhost:3000
   Network: http://192.168.x.x:3000
   ```
5. Отвори веб претраживач (Chrome, Safari, Firefox, Edge — било који ради)
6. Иди на **http://localhost:3000**

То је то! Игра ради!

### Соло режим

Кликни на **Соло Мисија**, унеси своје име и лозинку (да сачуваш резултат), и почни да играш!

- Изабери операције (сабирање, одузимање, или обоје)
- Изабери опсег бројева (1-10 за почетнике, до 1-100 за изазов)
- Користи брзе поставке: **Стартер**, **Класик**, **Изазов** или **Експерт**
- Укуцај одговор на тастатури и притисни **Enter**

### Режим за два играча (Свемирска трка)

Играј против некога на истој Wi-Fi мрежи!

1. Покрени игру на свом компјутеру (прати кораке горе)
2. Кликни на **Свемирска Трка** на свом компјутеру — видећеш мрежну адресу попут `http://192.168.x.x:3000`
3. На уређају другог играча (телефон, таблет или други компјутер), отвори претраживач и укуцај ту адресу
4. Оба играча су сад повезана! Домаћин кликне **Покрени Игру**

---

## Како да зауставиш игру

У терминалу где игра ради, притисни **Ctrl + C** да је зауставиш.

---

## Решавање проблема

**"command not found: node" или "command not found: npm"**
Node.js још није инсталиран, или треба да рестартујеш терминал после инсталације. Врати се на Корак 1.

**"address already in use"**
Игра већ ради у другом терминалу. Прво затвори тај терминал, или притисни Ctrl + C у њему.

**Други играч не може да се повеже**
Провери да ли су оба уређаја на истој Wi-Fi мрежи. Користи Network адресу приказану у терминалу (не `localhost`).
